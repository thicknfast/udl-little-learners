## Goblin Tools Video Walkthrough

Watch this video walkthrough explaining how Goblin Tools can be used to break down tasks and support executive function during transitions. Goblin Tools is a free, AI-powered tool that helps make multi-step tasks more manageable.

### How Goblin Tools Supports UDL

- Breaks complex tasks into smaller, more manageable steps
- Provides visual structure for transitions and routines
- Supports students who struggle with executive function and sequencing
